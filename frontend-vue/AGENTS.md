## Codex 协作规则

- 每次任务必须小步修改，一次只完成一个明确目标。
- 默认不安装依赖。
- 默认不执行 `npm audit fix --force`。
- 未经确认不得修改 `package.json`、`vite.config.js`、`main.js`。
- 不访问项目目录外文件。
- 中文文件读取时注意编码，不要因为 PowerShell 显示乱码就判断文件损坏。
- 每次修改后需要说明改了哪些文件，并建议运行 `npm run build`。
